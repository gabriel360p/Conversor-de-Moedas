# 💱 Conversor de Moedas

Um projeto simples e funcional de conversão de moedas, desenvolvido com foco em praticar **JavaScript**, manipulação de DOM e lógica de programação.

## 📌 Sobre o projeto

O **Conversor de Moedas** permite ao usuário inserir um valor em **Real Brasileiro (R$)** e convertê-lo para outras moedas de forma rápida e intuitiva.

As conversões são feitas com base em cotações previamente definidas no código.

## 🚀 Funcionalidades

* ✅ Conversão de Real para:
  * Dólar (US$)
  * Libra (£)
  * Euro (€)
  * Bitcoin (₿)
* ✅ Atualização dinâmica do valor convertido
* ✅ Alteração automática da bandeira da moeda selecionada
* ✅ Validação de input (evita conversão sem valor)
* ✅ Interface simples e responsiva

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🧠 Conceitos aplicados

* Manipulação do DOM (`querySelector`)
* Eventos (`addEventListener`)
* Estrutura de dados com objetos
* Condições (`if/else`)
* Template strings

## ▶️ Como executar o projeto

1. Clone o repositório:


```bash
cd conversor-de-moedas
```

3. Abra o arquivo `index.html` no navegador

## ⚠️ Observações

* As cotações das moedas são **fixas**, definidas diretamente no código.
* O projeto não utiliza API externa (ainda 👀).
